Break the ties among all the projects provided in input and returns them ordered. The tie-breaking can be based on the instance or/and on the profile. :param instance: The instance. :type instance: Instance :param profile: The profile. :type profile: pabutools.profile.Profile :param projects: The set of projects between which ties are to be broken. :type projects: collection of pabutools.election.instance.Project :param key: A key function to select the project from the input. :type key: function[_ -> pabutools.election.instance.Project]

Break the ties among all the projects provided in input and returns a single project. Orders the projects according to the tie-breaking rule and return the first project of the order. :param instance: The instance. :type instance: Instance :param profile: The profile. :type profile: pabutools.profile.Profile :param projects: The set of projects between which ties are to be broken. :type projects: collection of pabutools.election.instance.Project :param key: A key function to select the project from the input. :type key: function[_ -> pabutools.election.instance.Project]

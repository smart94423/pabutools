class AdditiveSatisfaction(instance: Instance, profile: Profile, ballot: ApprovalBallot, func: Callable[[Instance, Profile, ApprovalBallot, Project], Number])[source]#

Class representing additive satisfaction functions, that is, satisfaction functions for which the total satisfaction is exactly the sum of the satisfaction of the individual projects. To speed up computations, scores are only computed once and for all.

Parameters:

  • instance (Instance) – The instance.

  • profile (pabutools.instance.profile.Profile) – The profile.

  • ballot (pabutools.instance.profile.Ballot) – The approval ballot.

  • func (Callable[[PBInstance, Profile, Ballot, Project], Number]) – A function taking as input an instance, a profile, a ballot and a project, and returning the score as a fraction.

func#

A function taking as input an instance, a profile, a ballot and a project, and returning the score of the project as a fraction.

Type:

function

sat(projects: Iterable[Project]) → Number[source]#

Returns the satisfaction of a voter with a given approval ballot for a given subset of projects. The satisfaction is additive, it is thus defines as the sum of the score of the projects under consideration.

Parameters:

projects (set of Project) – The set of projects.

Return type:

float
